# QRCodeScanner

基于 [zxing-cpp](https://github.com/zxing-cpp/zxing-cpp) 库的一维码、二维码识别与生成工具，支持打开文件与相机采集方式识别，识别后标记识别的区域。

zxing-cpp通过提供头文件 [ZXingQtReader.h](https://github.com/zxing-cpp/zxing-cpp/blob/master/example/ZXingQtReader.h) 实现Qt接口的转换。

内置zxing-cpp最新2.3.0自编译库（VS 2022）

已测试环境：VS2022 + Qt6.8.2

兼容Qt5.15.2
